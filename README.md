# Foundations of Programming course @ IST

**Project 1: MNK Board Game**

Goal: Build a Python program that lets a human play an m‑n‑k style board game against the computer. The project
focuses on creating the full game logic from scratch - representing the board, checking wins in all
directions, handling turns, and implementing three computer difficulty levels. A mix of clean function design, simple
heuristics, and practical problem‑solving to make the game feel responsive and fun to play.

**Project 2: Orbito‑n Game**

Goal: Create a Python program that plays an adapted version of the board game Orbito, against the computer or another
player. The project focuses on modeling the game board using abstract data types, handling the unique “place and
rotate” mechanics, detecting winning lines, and implementing an automatic strategy that chooses smart moves based on
the board state. It’s a mix of game logic, careful state updates, and heuristics to make the computer feel like a
real opponent.
